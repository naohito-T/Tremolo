## envについて

デフォルトでnextがよしなに、NODE_ENVを設定してくれる

>より大きなReactエコシステムはNODE_ENV慣例として扱い、3つの値のみを許可します。
>production：アプリケーションがで構築されている場合next build
>development：アプリケーションがで実行される場合next dev
>test：アプリケーションがテストされているとき（例jest）
